Adding Experiments

Fluorescence data can be added to qPCRtoolkit as files in RDML format (.rdml or .lc96p). To add files click Choose File at Experiments tab. Multiply files can be uploaded (one by one) for usage at one analysis. Run data of each uploaded file is listed at Experiments tab. Unchecking of this data is results in excluding corresponding experiment from analysis. Samples conditions for experiments from devices except LightCycler 96 should added to samples names as SAMPLE_NAME@CONDITION.

Click Example File to load example which can be used as example for absolute and relative quantification.

Performing Absolute Quantification

This analysis can be done on the Absolute Quantification tab.

Sidebar panel:

  • Limit Cycles — limits min and max cycle to analyse. All fluorescence points below min cycle and upper max cycle will be omitted. If one of experiments has less cycles than max cycle then fluorescence data point will be added (same as last available).

  • Background Cycles — min and max cycle to use as background (is displayed on the Amplification Plot as gray vertical bar).

  • Apply Background Substraction — use or do not use background substraction.

  • Threshold TARGET_NAME — threshold level for each target in analysis (is displayed on the Amplification Plot as horisontal lines colored as targets).

  • Show — selects type of plot to view: amplification of calibration.

  • Targets — selects which targets will be shown.

  • Export .xlsx — exports all data from Results Table as Excell file.

Main panel:

  • Amplification or calibration plot

  • Amplification Plot — shows amplications curves colored by type of sample or target.

  • Calibration Plot — shows calibration curves. And results of efficiency calculations.

  • Plate View — shows plate for each experiment. Clicking on cell selects curve to view.

  • Results Table — represents information abput samples and calculated values.

Performing Relative Quantification

Absolute quantification should be performed before relative quantification.

Sidebar panel:

  • Reference Genes — selects reference genes to use in analysis.

  • Run Calibrator — selects sample to use as run calibrator — calibrator between experiments (select Cal for example file).

  • Study Calibrator — selects condition to use as calibrator inside analysis (select 0h for example file)

  • Show Targets — which targets will be shown on Bar Plot.

  • Show Samples — which samples will be shown on Bar Plot.

  • Show Conditions — which conditions will be shown on Bar Plot.

  • Split By — defines in which order results on Bar Plot will be separated.

  • Show Ratio — selects type of analysis to show on Bar Plot: Relative, Normalized (Run Calibrator should be defined), Scaled (Study Calibrator should be defined).

  • Y-Axis Type — log or linear y-axis.

  • Show Errors — show errors of calculation on Bar Plot if checked.

  • Export .xlsx — exports all data from Results Table as Excell file.

Main panel:

  • Bar Plot — mean values of calculated ratios.

  • Results Table — represents information abput samples and calculated values.
